Framework Overview

Optimized Performance

Built with ox_lib and oxmysql for maximum efficiency and minimal resource usage.

Modular Design

12 integrated resources working seamlessly together, each handling specific functionality.

Centralized Config

All settings managed through convars.cfg for easy server configuration.

Built-in Security

Comprehensive anti-cheat system with progressive punishment and admin bypass.

Modern UI

Purple-themed interface across all resources with customizable settings.

Developer Friendly

Clean API, extensive exports, and comprehensive documentation.

Installation

1

Install Dependencies

ensure oxmysql
ensure ox_lib
2

Import Database

Import as-core/database.sql into your MySQL database

3

Add to server.cfg

ensure as-core
exec @as-core/convars.cfg
ensure as-spawn
ensure as-target
ensure as-hud
ensure as-fuel
ensure as-anticheat
ensure as-admin
4

Configure Settings

Edit as-core/convars.cfg to customize framework settings
